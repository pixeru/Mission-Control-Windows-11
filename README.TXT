Emcee.Shareware 1.7 (x64)
README FOR EMCEE SHAREWARE
--------------------------

THE OTHER FILES INCLUDED IN THIS .ZIP INCLUDE CODE, IMAGES, ETC., AND
CONFIGURATION FILES NEEDED TO BUILD THE PROGRAM UNDER VISUAL STUDIO.
THE PROJECT NAME IS "EMCEE.SHAREWARE"  

IT WILL BUILD WITH VISUAL STUDIO 2013 COMMUNITY (THE FREE VERSION), 
BUT  BY DEFAULT VISUAL STUDIO 2013 DOESN'T ACCESS THE WINDOWS 10 
SDK, MEANING THAT THE SUPPORT FOR MULTIPLE DESKTOPS WON'T WORK: 
AND CHANGING THE SDK CAN BE TRICKY IN 2013 COMMUNITY.  THIS IS THE VERSION
IN VERSION 1.2.

VERSION 1.3 CAN WILL COMPILE AND RUN FROM VISUAL STUDIO 2015 COMMUNITY, 
WHICH USES THE WINDOWS 10 SDK, RESTORING THE MULTIPLE-DESKTOP SUPPORT.
IT IS NECESSARY TO SET THE PROJECT TO COMPILE USING THE ISO C++ 14 STANDARD 
IN THE PROJECT PROPERTIES AT

     Configuration Properties->C/C++->Language->C++ Language Standard
     
VERSION 1.7 CAN COMPILE WITH VISUAL STUDIO 2017 COMMUNITY (FREE FROM MS) AS AN
X64 APPLICATION.

THE CODE IS DESIGNED TO OPERATE CORRECTLY UNDER WINDOWS 10.   IT SHOULD
FUNCTION OK UNDER WINDOWS 7, BUT NOT VERY WELL UNDER WINDOWS 8 (WHO USES WINDOWS 8 ANYMORE?)
MODIFICATIONS MAY BE REQUIRED UNDER THOSE OLDER OS VERSIONS OR UNDER FUTURE VERSIONS
OF WINDOWS 10. 

BON APPETIT!
